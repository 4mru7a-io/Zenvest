# 🚀 Zenvest Quick Start Guide

## 60 Seconds to Your First Stock Trade

### Step 1: Run the App (30 seconds)
```bash
cd zenvest
npm install
npm run dev
```

### Step 2: Try Demo Mode (10 seconds)
- Open `http://localhost:5173`
- Click **"Try Demo Mode"** button
- No login needed! ✅

### Step 3: Buy Your First Stock (20 seconds)
1. Go to **Explore** (📊 tab)
2. Click any stock (e.g., TCS)
3. Click **🛒 Buy Stock**
4. Enter quantity (e.g., 1)
5. Click **Confirm Buy** ✅

🎉 **Congratulations!** You just made your first trade!

- Get answers in simple English

### 6. Check Your Profile (1 min)

- See your stats

- Track your progress

## 🐛 Troubleshooting

### Issue: App not loading
**Solution:**
1. Hard refresh: `Ctrl+Shift+R`
2. Clear cache: Settings → Clear browsing data
3. Try different browser
4. Check internet connection

### Issue: Trades not saving
**Solution:**
1. Refresh page (F5)
2. Check browser console (F12)
3. Restart dev server
4. Try demo mode again

### Issue: Charts not showing
**Solution:**
1. Wait 2-3 seconds for load
2. Refresh page
3. Check browser zoom (should be 100%)
4. Try different screen size

### Issue: Quiz answers not working
**Solution:**
1. Try different quiz
2. Check internet connection
3. Refresh page
4. Clear browser cache
